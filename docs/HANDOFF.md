# מסמך העברה / זיכרון פרויקט: אתר המועמדות של חן דגן

מסמך זה מסכם את מצב הפרויקט כדי שכל צ'אט או אדם שימשיך יוכל להיכנס לעניינים מהר. עודכן לאחר ההשקה החיה.

## מה זה
אתר מועמדת + תיק עבודות לחן דגן, שממצב אותה לתפקיד **ראשון בפלנינג ואסטרטגיה שיווקית במשרדי פרסום**. לא אתר שמוכר שירותי צילום. בעיית התפיסה שהאתר פותר: "היא רק יוצרת תוכן" → "מועמדת אסטרטגית עם תהליך עבודה מוכח".

## כתובות
* **אתר חי:** https://hen-dagan.vercel.app
* **GitHub:** https://github.com/hendagann/hen-dagan (ענף `main`)
* **Vercel:** scope `hen-dagan-s-projects`, פרויקט `hen-dagan`, תוכנית Hobby. **דחיפה ל־main = דיפלוי אוטומטי לפרודקשן.**

## מיצוב מאושר (אחרי מועצת llm-council + אישורי חן)
* טייטל: **חן דגן | פלנינג ואסטרטגיה שיווקית** (בלי "מועמדת", בלי "תוכן", בלי "מבוססת מחקר" ככותרת).
* מסר מרכזי: מזהה את המתח שמאחורי ההתנהגות ובונה ממנו כיוון מסרים, מוכח על לקוחות אמיתיים.
* מיקוד: משרדי פרסום בלבד (לא הייטק/PMM). עברית first + עמוד `/en` יחיד.

## סטאק וארכיטקטורה
* Next.js 15 (App Router) + TypeScript + Tailwind v4 + React Three Fiber + Liquid Glass (CSS).
* עברית RTL עם logical properties. פונט **Assistant בלבד**. עיצוב Premium דארק (ink `#0b0e17`, זהב `#e2b45a`, סגול `#8b7cf6`).
* **מקור האמת לכל הקופי: `content/site.ts`.** לא לערוך טקסט בקומפוננטות; לערוך שם.
* עמודים: `/` (בית), `/work` + `/work/[slug]` (chatwithus, glowguard, amg, smile), `/process`, `/about`, `/capabilities`, `/contact`, `/en`.
* קומפוננטות מפתח: `Nav` (רספונסיבי: מלא בדסקטופ, קומפקטי + המבורגר כשצר), `ThinkingScene` (סצנת חלקיקים תלת־ממדית), `DeckCarousel` (קרוסלת שקפים/המלצות), `OriginStory` (סקשן הצבא), `Reveal`, `VideoPlayer`, `Footer`.

## כללי ברזל לעריכה (אמינות + סגנון)
1. **אפס תווי מקף (־ – — -) בכל קופי האתר.** הרחבה בנקודתיים בלבד. (יש לבדוק אחרי כל שינוי טקסט.)
2. **אמינות מוחלטת.** אין להמציא תוצאות/לקוחות/מחקר/בעלות.
3. **לא להזכיר את לירן (צלם) ואת אליה (שותפה ל־GlowGuard)** בשמם. GlowGuard מוצג כפרויקט זוגי.
4. **"פי שלושה לידים" שייך לקמפיין AMG קודם בלבד** (לפי דיווח הלקוח), לא לסדרת הגיוס.
5. **הצבא: לכתוב "תעודת תא״ל" בלבד.** אסור "קצין/קצינה בדרגת תת אלוף".
6. StandWithUs/ChatWithUs: מסגור אקדמי נייטרלי, בלי הצהרת עמדה.
7. תוצאות תמיד "לפי דיווח הלקוח". פרויקטים אקדמיים מסומנים ככאלה.

## נכסים (איך הופקו)
* מקור: `~/Downloads/קלוד כנס לכאן` (וגם `~/Downloads/ממליצים `, רווח נסתר בסוף). הכל הועתק ל־`public/media`.
* **וידאו:** נדחס מ־446MB ל־~149MB עם Swift/AVFoundation (אין ffmpeg במחשב; סקריפט `render/compress.swift` היה ב־scratchpad, לא בריפו). כל קובץ < 100MB.
* **שקפי מצגות + כתבת עיתונות:** רונדרו לתמונות עם Swift/PDFKit (poppler לא מותקן).
* המצגות המלאות זמינות להורדה כ־PDF מעמודי הקייסים.

## הערות דיפלוי (תקלות שנפתרו, לידיעה)
* אם דף מחזיר 404 בפרודקשן בעוד `/media/*` נטען: ה־Framework לא זוהה כ־Next.js. יש `vercel.json` עם `framework:nextjs` שמתקן זאת.
* Deployment Protection ב־Vercel חוסם צפייה ציבורית — צריך להיות Disabled (Settings → Deployment Protection).
* אין push/פריסה בלי אישור מפורש של חן.
